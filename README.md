# AllNetworks website

Static site for AllNetworks – top up, SIM & eSIM orders.

**This folder is the one to upload to Git.** It contains only the website (no verification server).

## Contents

- **index.html** – Homepage and order form
- **terms.html** – Terms of Use & Privacy Policy
- **about-allnetworks-english.html** – About Us
- **vercel.json** / **.vercelignore** – Vercel deploy config

## Deploy to Vercel

See **README_DEPLOY.md** for step-by-step instructions (CLI or Git).

Quick: from this folder run `vercel` (after `npm i -g vercel`).
